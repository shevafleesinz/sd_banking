## ✨ Features

- 💳 Clean and modern bank interface
- 🏦 Withdraw, Deposit, and Transfer functions
- 👤 Auto-display of player name on transfers
- 🧾 Transaction log system
- ✅ ESX-only (Legacy and extended supported)
- 📱 Responsive UI for all screen sizes
- ⚡ Built with HTML, TailwindCSS, JavaScript

---

## 📦 Installation

1. Extract the resource to:
```

resources/sd_banking

```
2. Add this to your `server.cfg`:
```

ensure modern\_bank

```
3. Make sure `ox_lib` is properly installed and configured.

---

## ⚙️ Configuration

- Adjust framework export settings in `config.lua` if needed.
- To change UI style (colors, spacing), edit `html/style.css`.

---

## 🔧 Dependencies

- [ESX Framework](https://github.com/esx-framework/esx_core)
- [ox_lib](https://github.com/overextended/ox_lib)
